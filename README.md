# Yes Bank Stock Price Prediction

## Project Overview

This project focuses on predicting the monthly closing stock price of Yes Bank using Machine Learning techniques. The dataset contains historical stock price information, including Open, High, Low, and Close prices. The objective is to build a predictive model that can accurately forecast the closing stock price based on historical market data.

Yes Bank has experienced significant fluctuations in its stock prices due to various financial events and market conditions. Through Exploratory Data Analysis (EDA), feature engineering, and machine learning modeling, this project aims to identify important patterns and provide data-driven insights.

---

## Problem Statement

The goal of this project is to predict the monthly closing stock price of Yes Bank using historical stock market data. Accurate stock price prediction can help investors, analysts, and financial institutions make informed decisions and better understand market behavior.

---

## Business Objective

The business objective is to analyze historical stock price trends and develop a predictive model that can estimate future closing prices. The insights generated from this analysis can support investment decisions, risk assessment, and financial planning.

---

The dataset contains the following features:

* Date
* Open Price
* High Price
* Low Price
* Close Price (Target Variable)

## Machine Learning Models Used

### 1. Linear Regression

A baseline regression model used for stock price prediction.

### 2. Decision Tree Regressor

Used to capture non-linear relationships in stock price data.

### 3. Random Forest Regressor

An ensemble learning algorithm used to improve prediction accuracy and reduce overfitting.

---

## Hyperparameter Tuning

GridSearchCV was used to optimize model performance by identifying the best hyperparameter combinations through cross-validation.

---

## Evaluation Metrics

The models were evaluated using:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score

These metrics were used to assess prediction accuracy and model performance.

---

## Feature Importance

Feature importance analysis was performed using Random Forest Regressor to identify the most influential factors affecting the closing stock price.

Key features included:

* Open Price
* High Price
* Low Price

---

## Results

Among all models, the Random Forest Regressor achieved the best performance with higher prediction accuracy and better generalization on unseen data.
